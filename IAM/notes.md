## Customer Managed Policy

Policies created and maintained by AWS account administrators.

Advantages

- Reusable
- Version Controlled
- Custom Permissions

---

## IAM Groups

A collection of IAM users that share identical permissions.

Benefits

- Easier Permission Management
- Centralized Administration

---

## Policy Simulator

Allows administrators to test IAM permissions before applying them.

---

## Access Advisor

Shows which AWS services have actually been used by an IAM identity.

Useful for

- Removing unused permissions
- Security Auditing

---

## Permission Evaluation

AWS evaluates requests using:

1. Authentication
2. Identity-based Policies
3. Resource Policies
4. Explicit Deny
5. Allow
6. Default Deny
